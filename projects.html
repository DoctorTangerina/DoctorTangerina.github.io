Random Event Dice Roller (W.I.P)


Este projeto é um plugin para servidores de minecraft, feito no Eclipse utilizando a linguagem Java.
Esse plugin gera um número de 1 a 20 aleatóriamente, e dependendo do número sorteado, efetua um evento aleatório.
